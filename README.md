# Himeko-Manga-App

Created a manga app using MERN stack.

### Features:
```
  1. User Authentication
  2. User Dashboard (Read History)
  3. Huge Manga library where you can search any manga you want.
  4. Attractive UI
```

## Setup

 **!!Pre-requisites**: NodeJS

##### Go to frontend dir
 ```
  cd ./FrontEnd
  npm install
  npm run dev
```

##### Go to backend dir
 ```
  cd ./BackEnd
  npm install
  npm start
```
